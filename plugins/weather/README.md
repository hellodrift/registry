# Weather

Weather forecast in the menu bar. Shows current temperature and conditions with a popover for 7-day forecasts and hourly breakdowns.

## Features

- Current temperature and conditions displayed in the menu bar
- 7-day forecast overview with daily highs/lows
- Hourly breakdown with precipitation probability, humidity, and wind speed
- Configurable location via geocoding search
- Fahrenheit and Celsius support
- Automatic timezone detection

## Setup

No API keys or credentials required. Uses the free [Open-Meteo](https://open-meteo.com/) API.

Configure your location in the plugin settings drawer.
