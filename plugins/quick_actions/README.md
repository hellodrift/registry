# Quick Actions

Run pnpm scripts from the menu bar with one click.

## Features

- Automatically discovers `package.json` scripts in your project directories
- Groups scripts by package for monorepo support
- Launches scripts in a terminal window
- Respects `.gitignore` rules during discovery
- Configurable project directories and worktree selection

## Setup

No credentials required. Open the plugin settings drawer to configure which project directories to scan for scripts.
