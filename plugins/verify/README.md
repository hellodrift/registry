# Verify Work

Read testing instructions and launch dev commands from the menu bar.

## Features

- Displays `testing-instructions.md` from your active workstream directories
- One-click launch of `pnpm run dev` in a terminal
- Offers to generate testing instructions via the workstream agent if none exist
- Renders markdown-formatted instructions inline

## Setup

No credentials required. The plugin automatically reads from your active workstream's directories. Add a `testing-instructions.md` file to any project directory to get started.
